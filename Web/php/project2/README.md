# Project 2: PHP Calculation & Forms

## Overview
แบบฝึกหัดพื้นฐาน PHP ในการรับค่าจาก HTML Form และนำมาประมวลผลทางคณิตศาสตร์

## Code Deep Dive

### 1. [`calcu.php`](./calcu.php) / [`calculate.html`](./calculate.html)
**Topic:** Calculator Logic
- รับค่าตัวเลขและเครื่องหมาย (+, -, *, /) จากฟอร์ม
- คำนวณผลลัพธ์ผ่าน PHP Script

### 2. [`feedback.php`](./feedback.php)
**Topic:** Data Handling
- รับความคิดเห็นจากผู้ใช้และแสดงผลทางหน้าจอ
- สาธิตการใช้ `$_POST` หรือ `$_GET` ในการดึงข้อมูล
