# System 2: Advanced Authentication

## Overview
เวอร์ชันต่อยอดของระบบ `System` โดยมีการปรับปรุงโครงสร้างและฟีเจอร์เพิ่มเติม

## Enhancements
- **Logout Logic**: การเคลียร์ Session อย่างถูกต้อง (`logout.php`)
- **Navigation**: เมนูบาร์ที่ปรับเปลี่ยนตามสถานะการล็อกอิน (`nav.php`)
- **Structure**: การแยกไฟล์ Style และ Content ที่ดีขึ้น
