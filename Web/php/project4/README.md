# Project 4: Form & Logic Applications

## Overview
รวมตัวอย่าง PHP Script ที่รับค่าจาก Form และประมวลผล Logic ที่ซับซ้อนขึ้น

## Scripts List
- **`game.php` / `goodgame.php`**: เกมทายตัวเลขหรือ Logic เกมง่ายๆ
- **`salary.php`**: โปรแกรมคำนวณเงินเดือนหรือภาษี
- **`profile.php`**: การแสดงผลข้อมูลส่วนตัวจากตัวแปร
- **`date.php`**: การจัดการวันและเวลา
