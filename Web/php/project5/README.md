# Project 5: File Loading and Utilities

## Overview
แบบฝึกหัดเกี่ยวกับการจัดการไฟล์และการคำนวณขั้นสูง

## Features
- **File Upload**: `upload.php` สาธิตการอัปโหลดไฟล์ (รูปภาพ) ไปเก็บที่ `uploaded_images/`
- **Calculation**: `area.php`, `cal1-3.php` โปรแกรมคำนวณพื้นที่และสูตรคณิตศาสตร์
- **Utilities**: `copyright.php`, `date.php` ฟังก์ชันช่วยเหลือทั่วไป
