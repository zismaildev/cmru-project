# Project 3: PHP Logic & Control Flow

## Overview
รวมตัวอย่างโค้ดการใช้ Logic พื้นฐานใน PHP เช่น Loop, Condition และการคำนวณต่างๆ

## script List
- **calcu/calculate**: โปรแกรมคำนวณเกรดหรือตัวเลข
- **compare**: การเปรียบเทียบค่า
- **multi_x**: ตัวอย่างการใช้ Loop แบบต่างๆ (`for`, `while`, `do-while`)
- **convertyear**: แปลงปี ค.ศ./พ.ศ.
- **showthai**: การจัดการภาษาไทย
