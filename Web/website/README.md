# Basic Website (Frontend)

## Overview
โปรเจกต์ฝึกหัดการสร้างเว็บไซต์ Frontend ด้วย HTML และ CSS พื้นฐาน 
- **Features**: การเชื่อมโยงหน้า (Links), การออกแบบ Layout ด้วย CSS, และฟอร์มรับข้อมูลเบื้องต้น

## File Structure
- **`index.html`**: หน้าแรกของเว็บไซต์ (Homepage) รวมเมนูนำทาง
- **`style.css`**: ไฟล์ตกแต่งหน้่าเว็บ (Stylesheet)
- **`profile.html`**: หน้าแสดงข้อมูลส่วนตัว
- **`login.html`**: แบบฟอร์มเข้าสู่ระบบ (Frontend UI)
- **`quiz.html`**: ตัวอย่างแบบทดสอบออนไลน์
