# Java Logic & Mathematics

## Overview
รวมแบบฝึกหัดด้านตรรกะและคณิตศาสตร์

## Projects Breakdown

### 1. Divisible Numbers
หาตัวเลขที่หารด้วยค่าที่กำหนดลงตัว

### 2. Number Stats
คำนวณค่าทางสถิติเบื้องต้น (เช่น ผลรวม, ค่าเฉลี่ย) จากชุดตัวเลข

### 3. SameFor (Sum Loop)
แบบฝึกหัดการใช้ `for-loop` เพื่อหาผลรวม (Summation)
```java
for (counter=1; counter <= 10; counter++) {
    sum += counter;
}
```

### 4. Switch Case Demo
การใช้งาน `switch-case` เพื่อควบคุม Flow ของโปรแกรมตามเงื่อนไข
