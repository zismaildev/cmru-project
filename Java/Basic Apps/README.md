# Basic Java Applications

## Overview
รวมโปรแกรมพื้นฐานสำหรับการคำนวณทั่วไปและสุขภาพ

## Projects

### 1. Calculator
โปรแกรมเครื่องคิดเลขพื้นฐาน (บวก ลบ คูณ หาร)

### 2. Health Calculator (BMI)
โปรแกรมคำนวณดัชนีมวลกาย (Body Mass Index)
- **Logic**: น้ำหนัก (kg) / ส่วนสูง (m)^2
- **Interpretation**: แปลผลลัพธ์ว่า ผอม, ปกติ, หรือ อ้วน
