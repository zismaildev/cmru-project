# Base 1 (พื้นฐาน Dart)

**ไฟล์:** [`base1.dart`](base1.dart)

## รายละเอียด (Description)
ไฟล์นี้รวบรวมตัวอย่างไวยากรณ์พื้นฐานของภาษา Dart ประกอบด้วย:
- **Operators**: การบวก ลบ คูณ หาร และการเปรียบเทียบค่าต่างๆ
- **Variables**: การประกาศตัวแปรประเภทต่างๆ เช่น `int`, `double`, `String`, `bool`
- **Collections**: การใช้งาน `List` และ `Map` เบื้องต้น
- **Constants**: การใช้ `const` และ `final`
