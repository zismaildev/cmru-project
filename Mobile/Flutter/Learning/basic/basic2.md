# Basic 2 (การควบคุม Flow)

**ไฟล์:** [`basic2.dart`](basic2.dart)

## รายละเอียด (Description)
ไฟล์นี้เน้นเรื่องการควบคุมการทำงานของโปรแกรม (Control Flow) ในภาษา Dart:
- **Loops**: ตัวอย่างการใช้ `for`, `while`, และ `do-while`
- **Conditions**: การใช้ `if-else` ในการตรวจสอบเงื่อนไข
- **Switch Case**: การตรวจสอบเงื่อนไขหลายทางเลือกด้วย `switch`
