# Assignment 1: Dart Basic Calculations

## Overview
แบบฝึกหัดการเขียนภาษา Dart เบื้องต้น เน้นการรับค่าจาก Input (stdin) และคำนวณทางคณิตศาสตร์

## Code Deep Dive

### 1. [`average.dart`](./average.dart)
**Topic:** Average Calculation
- รับค่าตัวเลขจำนวนเต็ม 3 ค่า
- คำนวณค่าเฉลี่ย `(num1 + num2 + num3) / 3`
- แสดงผลลัพธ์พร้อมทศนิยม

### 2. [`totaldistancetraveled.dart`](./totaldistancetraveled.dart)
**Topic:** Physics Calculation (Distance)
- รับค่าความเร็ว (Speed) และเวลา (Time)
- คำนวณระยะทางจากสูตร `Distance = Speed * Time`

### 3. [`volumeofcylinder.dart`](./volumeofcylinder.dart)
**Topic:** Geometry Calculation
- รับค่ารัศมี (Radius) และความสูง (Height)
- คำนวณปริมาตรทรงกระบอก `Volume = π * r² * h`
- ใช้ค่า π ≈ 3.14159
